#  Exercícios de Polimorfismo em C#

Este repositório contém dois exercícios práticos desenvolvidos com o objetivo de aplicar e compreender o conceito de **polimorfismo** na linguagem C#.

---

##  Sobre Polimorfismo

Polimorfismo é um dos pilares da Programação Orientada a Objetos (POO) e permite que métodos com o mesmo nome tenham comportamentos diferentes, dependendo do contexto em que são utilizados.

No C#, isso pode ser aplicado principalmente de duas formas:
- 🔹 **Sobrescrita (Override)** – utilizando herança
- 🔹 **Sobrecarga (Overload)** – métodos com assinaturas diferentes

---



##  Objetivo Geral

Os exercícios foram desenvolvidos para:
- Consolidar o entendimento de polimorfismo
- Praticar conceitos fundamentais de POO
- Aplicar teoria em exemplos práticos
